To get your own .obj file to try:
  • 1. Find an object you like on Thingiverse. Click on the Object, Download All Files, then unzip and go into Files to find the .stl file for the object on your computer.
  • 2. Use the Free Convert Tool to convert the .stl file to an .obj file by selecting Output Format (WaveFront Object (.obj)). Once converted, click the .obj file to download it to your computer for use by Dex-Net as a Service (DNaaS).
