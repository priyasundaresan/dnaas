Dex-Net as a Service

Upload object meshes and evaluate robotic grasps using Dex-Net

Ken Goldberg, Jeff Mahler, Pusong Li, Bill DeRose (UC Berkeley, AUTOLAB) and Juan Aparicio Ojea (Siemens)
Sponsored in part by Siemens, NSF, NDSEG, Google, and Autodesk

** Currently in beta release. Please report bugs and suggestions to Pusong Li at alanpusongli@berkeley.edu **

Dex-Net as a Service (DNaaS) is a prototype cloud-based grasp planning system. Anyone can upload a 3D object mesh (in .obj format with triangular faces) and visualize candidate grasp axes using a parallel-jaw gripper, each ranked by their robustness (green for most robust to perturbations). To try it, visit the Home page in the menu to the left.

Please note that computing grasps takes up to 3 minutes. The system is currently single-threaded and does not support simultaneous users. After grasps are computed they will be displayed as colored line segments. You may also choose to visualize grasps for each stable pose of your object. DNaaS is based on Dexterity Network 1.0 (Dex-Net), a data-driven approach to robust robot grasping and manipulation based on a dataset of 3D object models that includes over 10,000 unique 3D object models and 2.5 million parallel-jaw grasps. Dex-Net estimates the probability of force closure under sampled uncertainty in object and gripper pose and friction. Dex-Net 1.0 uses Multi-View Convolutional Neural Networks (MV-CNNs), a deep learning method for 3D object classification, as a similarity metric between objects.

For more information, papers, and videos, please see the Dex-Net project on Github.
